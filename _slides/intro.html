Instruction Set Architecture: How do we program a computer processor? 1. Microarchitecture: How is the processor internally implemented? 1. Performance: How do processors run code so fast? ___ ## Instruction Set Architecture (ISA) 1. Humans program processors using assembly language source code 1. Compilers generate machine code which is a binary representation of assembly language 1. You will learn how to program a processor using its ISA ___ ## Microarchitecture 1. Processors are electrical devices composed of wires and transistors 1. Transistors are used to build circuits to make memory and logic 1. You will learn how to build those circuits in a simulated environment ___ ## What are some examples of Architectures? 1. The most widely known architectures are Intel's x86 (PCs) and ARM (most mobile devices) 1. We will learn a new architecture called RISC-V 1. RISC-V is open source, and free to use --- # Syllabus and Policies 1. Course Style 1. Classroom Culture 1. Logistics 1. Notable Policies ___ ## Course Style 1. Learn by doing: labs and projects 1.
